# CircuitHub

电路学习与开源设计社区 — 从入门教程到仿真工具，从开源 PCB 项目到社区讨论，为每一位电路探索者构建的知识平台。

## 页面

| 路由 | 说明 |
|------|------|
| `/` | 首页：Hero 搜索 + 精选资源 + 社区动态 |
| `/resources` | 资源库：标签筛选 + 教程/仿真/PCB/开源/书籍 |
| `/community` | 社区：频道帖子流 + 问答 + 每周挑战 |
| `/projects` | 项目：共享组件库 + 开源项目 + Open Design |

## 本地开发

```bash
node server.js
# → http://localhost:3000
```

## 部署

通过 GitHub Pages 部署，源目录：`/docs`。

## 设计

- 暗色主题 · 极简排版
- Discord 风频道导航 + 论坛帖子流
- 响应式（桌面 / 平板 / 手机）
- 零依赖静态服务器
